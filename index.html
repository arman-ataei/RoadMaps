      1. Basics (operators, variables, Indentation, etc)
      2. Algorithm/PsudoCode for simple arithmetic problems (sum, prime numbers, etc)
      3. Conditional Statements (if-elseif-else, switch-case, etc)
      4. Loops (For, while, do-while, etc)
      5. Functions (single/multiple-input/output, single/multiple-input/output, recursive/none-recursive )
      6. Primitive DataStructures (bolean, int, float, char)
      7. Basic DataStructures (long, double, signed, unsigned, strings, arrays, stacks, queue)
      8. Simple Sorting Algorithms (bubble sort, insertion sort, etc)
      9. Some advanced sorting algorithm (merge sort, quick sort, etct)
      10. Structures and compound DataStructures
      11. Classes and encapsulations
      12. problem solving
      13. Advanced DataStructures (trees, graphs, sets, maps, etc) optional
    • C/C++
      1. Different versions of C/C++
      2. Structure of A C/C++ Program (headers, NameSpaces, main function, etc)
      3. Variables (int, float, etc)
      4. Operators & Operator precedence
      5. Functions (declaration/ definition)
      6. pointer & refrences
      7. Data Structures
      8. structures and clases
      9. conditional statements
      10. Loops
      11. Implementations of algorithms/psudocodes
